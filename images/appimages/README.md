# App Screenshots

这个目录需要包含应用截图，用于在 README.md 中显示。

## 需要的截图文件：

1. `应用主页截图.png` - 应用主界面截图
2. `设置长截图.png` - 设置页面截图
3. `自动翻译和自动摘要示意图.png` - AI 功能演示截图
4. `RSSHUB订阅支持景象设置.png` - RSSHub 配置界面截图
5. `支持自定义订阅的分类标签.png` - 标签分类功能截图
6. `展示时间过滤自动刷新订阅时间设定.png` - 时间过滤设置截图
7. `多语言支持中文.png` - 中文界面截图
8. `多语言支持英文.png` - 英文界面截图
9. `多语言支持日语.png` - 日文界面截图
10. `多语言支持韩文.png` - 韩文界面截图

## 添加方法：

1. 运行应用：`./start.sh`
2. 对应用进行截图
3. 将截图保存到此目录，使用上述文件名
4. 截图尺寸建议：宽度 360px 左右
5. 截图格式：PNG

注意：此目录在 .gitignore 中被忽略，截图不会提交到版本控制中。

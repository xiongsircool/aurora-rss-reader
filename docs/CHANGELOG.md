## [0.1.0] - 2025-01-13 🎉

### 🌟 首次发布
这是 Aurora RSS Reader 的第一个正式发布版本！

#### ✨ 核心功能
- 📖 **RSS 订阅管理**：添加、编辑、删除 RSS 源
- 🔄 **自动刷新**：定时获取最新文章内容
- 🏷️ **分组管理**：支持 RSS 源分类和标签
- 🎨 **三栏布局**：订阅列表、文章列表、内容预览
- 🔍 **全文搜索**：在文章标题和内容中搜索
- ⭐ **收藏功能**：标记和收藏重要文章
- 📖 **阅读状态**：标记已读/未读状态

#### 🌍 多语言支持
- 🇨🇳 **简体中文**
- 🇺🇸 **英语**
- 🇯🇵 **日语**
- 🇰🇷 **韩语**

#### 🤖 AI 增强功能
- 📝 **自动摘要**：使用 LLM 生成文章摘要
- 🌐 **智能翻译**：支持全文和标题翻译
- ⚙️ **可配置模型**：支持 OpenAI 兼容 API
- 💾 **智能缓存**：翻译结果本地缓存

#### 🌈 用户体验
- 🌓 **深色主题**：深色/浅色主题切换
- 📱 **响应式设计**：适配不同窗口大小
- 🔧 **布局调节**：可拖拽调整各栏宽度
- ⏰ **时间过滤**：按发布时间筛选文章
- 📊 **进度提示**：加载和刷新进度显示

#### 🔧 技术特性
- 💾 **本地数据库**：SQLite 本地存储
- 🚀 **跨平台**：支持 Windows、macOS、Linux
- 🌐 **RSSHUB 支持**：自定义 RSSHub 镜像
- 📤 **OPML 导入导出**：订阅数据备份和迁移
- 🔄 **后台服务**：后台自动刷新订阅

#### 🛡️ 安全和隐私
- 🔐 **本地优先**：数据主要存储在用户设备
- 🔑 **API 密钥安全**：敏感信息不提交到仓库
- 🛡️ **沙箱隔离**：Electron 安全最佳实践

## 版本说明

### 🏷️ 版本命名
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 🎨 图标说明
- 🌟 重要功能或重大更新
- 🆕 新增功能
- 🐛 问题修复
- 🔄 功能变更或改进
- 🗑️ 功能移除或弃用
- 🔒 安全相关更新
- 🌍 国际化更新
- 📚 文档更新
- ⚡ 性能优化
- 🎨 界面改进

### 📋 发布周期
- **主版本**：根据重大功能需求发布
- **次版本**：每月底定期发布
- **修订版本**：根据问题修复情况发布
